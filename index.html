AI Policy Explorer

GitHub Pages-barát miniapp: lekérdezés (checkboxos párbeszédablak), táblázat (max 3 szemponttal), és adatok (JSON/CSV + policy URL-ek).

GitHub gyors telepítés
  1. Hozz létre egy repo-t, pl. ai-policy-explorer.
  2. Másold be a fájlokat a repo gyökerébe: index.html, app.js, style.css, data.json, data.csv.
  3. Repo Settings → Pages → Deploy from branch → main / root.
